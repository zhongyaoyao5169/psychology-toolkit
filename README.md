# 心理学工具集

课程配套的心理学可视化与小工具合集，统一由此入口进入各子项目。

## 子项目

| 目录 | 说明 |
| --- | --- |
| [心理学名人堂](./心理学名人堂/) | 学者人物脉络图，纵轴年代、横轴学派与研究方向 |
| [心理学统计数理](./心理学统计数理/) | 统计计算与可视化演示（含 Plotly 与 GIF 动画） |
| [心理学实验](./心理学实验/) | 经典实验图文详解，支持目录切换与搜索 |
| [心理学测验](./心理学测验/) | 临床常用智力与人格量表介绍与样题试测 |

## 使用方式

1. 打开根目录 [`index.html`](./index.html) 进入工具集首页。
2. 点击卡片进入对应子项目。
3. 多数页面为纯静态 HTML，直接打开即可使用。

### 心理学统计数理（含 GIF）

在 `心理学统计数理` 目录运行本地服务：

```bash
python3 serve_web.py
```

## 数据构建

部分子项目的数据由 Python 脚本生成：

```bash
# 心理学实验
cd 心理学实验 && python3 build_data.py

# 心理学测验
cd 心理学测验 && python3 build_data.py
```
